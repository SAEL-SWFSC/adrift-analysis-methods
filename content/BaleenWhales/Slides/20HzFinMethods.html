Detecting 20Hz fin pulses in PAMGuard

  • Currently we are using PAMGuard version 2.02.09.
  • We used the click detector to identify fin pulses and classify them at 20Hz clicks

Step 1: Open PAMGuard

  • Open PAMgaurd and navigate to the settings file that has FinClick in the name

Step 2: Change output settings for your Drift

  • Load a new database with the following naming structure

    • ADRIFT_###_PG_2_02_09_FinClick.sqlite3

  • Setup a new Binary files folder

    • We want each Drift to have its own Binary files
